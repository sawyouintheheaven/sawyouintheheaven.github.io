vscode-mindmap-输出bug

2020年05月07日 270 字 大概 1 分钟
  1. 1. 先说结论
  2. 2. 解决方法
  3. 3. detail
  4. 4. 解决bug可能不难，但是找到bug却没那么容易

  • 先说结论

    vscode 缩放导致插件的ui没显示出来，进而不能输出图片

  • 解决方法

    ctrl - 缩放，或者把左边EXPLORE隐藏即可

  • detail

    接触vscode之后其他什么app都想使用插件，脑图也不例外，用的是vscode-mindmap。今天给导师演示脑图，末导师想让我输出图片给他,结果开始折腾。

    • 插件那里是这么说的

      可是我怎么找也找不到输出选项，默认的File和Edit应该是属于vscode的，插件应该不会进入这些选项卡。
      国内搜索引擎试探一番，未果。尝试google，结果找到一个日本小哥写的帖子
      文末有一张图片我很在意：
      为什么它的ui里有这个选项？然后我又去查了这个插件的历史版本，似乎也有。难道最新版给去掉了？不可能吧？
      于是突然意识到是vscode没显示全的问题，缩放一下舒服了。
